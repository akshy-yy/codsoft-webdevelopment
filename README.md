# 💼 CodSoft Web Development Tasks

This repository contains three beginner-to-intermediate level web development projects built using **HTML, CSS, and JavaScript**. These projects demonstrate fundamental front-end development skills along with clean UI design and structured implementation.

---

## 📌 Projects Included

### 1️⃣ Portfolio Website

A personal portfolio website showcasing:

* About section with profile details
* Skills and technologies
* Project highlights
* Resume download option
* Contact information

---

### 2️⃣ Landing Page

A professional and modern landing page designed for a tech product:

* Clean navigation bar
* Hero section with call-to-action
* Feature highlights
* About section
* Structured layout and responsive design principles

---

### 3️⃣ Calculator

An interactive calculator application with:

* Basic arithmetic operations
* Clean and minimal UI
* Responsive button layout using CSS Grid
* JavaScript-based logic handling

---
